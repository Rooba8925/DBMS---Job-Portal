Project Overview

The Hospitality Forecast Dashboard is a web-based application designed to analyze hospitality-related data and provide insights into hotel performance. The system allows administrators to log in and view a dashboard that helps in understanding customer trends, hotel services, and revenue-related insights.

The project demonstrates how a web application can be used to manage and visualize hospitality data effectively.

Objectives

Develop a web-based dashboard for hospitality data analysis
Provide secure admin login authentication
Display hospitality data in an organized dashboard interface
Help analyze customer trends and hotel performance

Technologies Used

HTML – For designing the web pages
CSS – For styling the interface
PHP – For backend logic and authentication
MySQL – For database management
XAMPP – For running the local server

Project Features

Admin Login and Authentication System
Secure password verification
Dashboard for viewing hospitality data
User session management
Logout functionality
